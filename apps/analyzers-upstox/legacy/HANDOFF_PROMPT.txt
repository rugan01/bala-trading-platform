Important operating rules:
- This project has three analyzers: mcx_monitor, stock_fo_monitor, and stock_intraday_monitor.
- Use Upstox live quote API as the source of truth for real-time checks.
- Be careful: the Upstox 15m historical endpoint for stocks can return stale prior-session candles intraday.
- Do not trust exact intraday entry/SL/target levels unless the latest 15m candle is from today.
- For live option-position checks, always resolve the exact strike and expiry from the Upstox instrument master before checking quotes.
- If an option book is wide or illiquid, manage the trade primarily from the underlying spot levels rather than only the option prices.
- Do not move files or change the output folder structure unless necessary.
